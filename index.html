MARBLE MAZE

Instructions

Tilt the board to guide the marble to the finish! Be careful to avoid falling into the traps!

Controls

  • [W] - tilt board up

  • [S] - tilt board down

  • [A] - tilt board left

  • [D] - tilt board right

  • [R] - reset game

A JavaScript and HTML Canvas game
